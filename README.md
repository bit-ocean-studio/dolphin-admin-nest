# Dolphin Admin Nest

English / [简体中文](./README.zh-CN.md)

Dolphin Admin Nest is the back-end service of Dolphin Admin, based on Nest.js, TypeScript, Prisma and PostgresSQL.

## Tech Stack

- Nest
- TypeScript
- Prisma
- PostgreSQL

## Usage

### Install

```bash
pnpm i
```

### Build

```bash
pnpm build
```
