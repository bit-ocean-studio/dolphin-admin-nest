# Dolphin Admin Nest

[English](./README.md) / 简体中文

Dolphin Admin Nest 是 Dolphin Admin 的后端服务，基于 Nest.js、TypeScript、Prisma 和 PostgreSQL。

## 技术栈

- Nest
- TypeScript
- Prisma
- PostgreSQL

## 特性

- 本地文件服务
- 腾讯云对象存储 COS 服务

## 使用

### 安装

```bash
pnpm i
```

### 启动

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```
